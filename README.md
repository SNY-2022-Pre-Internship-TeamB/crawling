# crawling

## 목표
온라인 청년센터의 주거/금융 정책을 크롤링

## 사용법
- Repository Clone
```shell
git clone https://github.com/whquddnr97/crawling.git
```

- Set directory
```shell
cd crawling
```

- Install requirements
```shell
pip install -r requirements.txt
```

- Run
```shell
python3 main.py
```

## 결과
- 작업 디렉토리에 csv파일로 저장됨

- Excel로 열 시, 인코딩을 변경해 주어야 함
